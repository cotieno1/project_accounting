1. Start BOM — add BOM line items (item, UOM, qty).
  2. Save / done — lock the number of items in a BOM and submit when your list is complete. Your Snr Site Engineer task on this screen is finished at that point.
  3. Print BOM — the request from the technical on qty and item type or specification (BOM List → PDF). Used to raise a requisition order (RO Number) with one-to-one items as in the BOM.
  4. Next step — raise requisition — confirm and request in memo to the GM; print RO and, on GM approval, print an RFQ and send to the supplier.
  5. View MRO details — when approved, open your MRO. For a partial buy, select the line (e.g. B1) and the quantity you need now.
  6. BOM view — prints for the Snr Site Engineer to sign; sent to the GM to start the budget and procurement process.
